# Telegram-бот «Всё о ДСЖ»

## Что уже готово

- 12 новых вопросов и правильные ответы.
- Отдельное изображение с разными персонажами для каждого вопроса.
- Подсчёт баллов и рейтинг.
- Админ-панель.
- Редактирование текста, вариантов ответа, правильного ответа и изображения прямо в Telegram.
- База игроков: Telegram ID + имя + фотография.
- Автоматическая подстановка фото победителя по сохранённому ID.
- Если фото не сохранено, бот пытается получить аватар Telegram.
- Кнопка «Завершить игру» показывает победителя, его фото, баллы и Telegram ID.

## Быстрый запуск

1. Создайте бота через @BotFather и скопируйте токен.
2. Узнайте свой Telegram ID через @userinfobot.
3. Скопируйте `.env.example` в `.env`.
4. Вставьте токен и свой ID:
   ```
   BOT_TOKEN=123456:ABC...
   ADMIN_IDS=123456789
   ```
5. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
6. Запустите:
   ```
   python main.py
   ```

## Как заранее добавить игрока и фотографию

1. Откройте «⚙️ Админ-панель».
2. Нажмите «👥 Игроки».
3. Нажмите «➕ Добавить игрока».
4. Отправьте Telegram ID.
5. Введите имя.
6. Отправьте фотографию.

Когда этот ID займёт первое место, бот автоматически покажет сохранённую фотографию.

## Как изменить вопрос

1. Откройте «⚙️ Админ-панель».
2. Нажмите «❓ Вопросы».
3. Выберите вопрос.
4. Измените текст, четыре варианта, правильный ответ или изображение.

## Важно

Участник должен хотя бы один раз открыть бота и пройти тест. Для получения аватара Telegram доступ к фото профиля должен быть разрешён настройками пользователя.
